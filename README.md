# Notice Board

A simple Notice Board application built using Next.js, Prisma, and TiDB Cloud. Users can create, view, update, and delete notices.

## Tech Stack

- Next.js
- JavaScript
- Prisma ORM
- TiDB Cloud
- CSS

## Features

- Add Notice
- View Notices
- Edit Notice
- Delete Notice
- Responsive UI

## Run Locally

Install dependencies:

```bash
npm install
```

Create a `.env` file and add your database URL.

```env
DATABASE_URL="your_database_url"
```

Generate Prisma Client:

```bash
npx prisma generate
```

Run the project:

```bash
npm run dev
```

Open:

```
http://localhost:3000
```
